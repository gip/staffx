title: "Operational Validation Checklist"
language: "en"
---

Use this checklist whenever validating StaffX runtime rebuilds. Confirm environment variables are set for API, web, and desktop. Confirm database migration applied. Confirm /v1/health is healthy. Confirm Auth0 login succeeds for web and desktop. Confirm project and thread CRUD paths function. Confirm matrix layout updates and events stream processing. Confirm assistant run queue claim, completion, cancellation, and message persistence paths. Confirm OpenShip bundle generation and reconciliation succeed for at least one thread.
