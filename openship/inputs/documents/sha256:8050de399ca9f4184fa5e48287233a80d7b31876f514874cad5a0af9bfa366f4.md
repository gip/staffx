title: "Features - Desktop Main Process"
language: "en"
---

The desktop main process owns native window lifecycle management, Auth0 PKCE login orchestration, secure token storage in memory, IPC command handling, and local direct assistant execution entrypoints. It coordinates OpenShip bundle fetch, local workspace synchronization, assistant run completion callbacks, and status propagation to renderer.
