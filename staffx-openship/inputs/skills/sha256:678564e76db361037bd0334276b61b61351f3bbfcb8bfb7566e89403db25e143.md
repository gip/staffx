Use Node.js with TypeScript. Structure: src/ with routes/, middleware/, services/, models/ subdirectories. Use express or fastify as HTTP framework. Use express-oauth2-jwt-bearer or equivalent for Auth0 JWT validation middleware. Environment configuration via dotenv.
